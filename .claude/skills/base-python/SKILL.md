## 読み込みタイミング

### パッケージマネージャー

新規プロジェクトおよびすでに `uv` を使用しているプロジェクトでは `uv` をデフォルトとする。
`uv` がインストールできないレガシー環境のみ `venv` + `requirements.txt` にフォールバックする。

### スタイル

`standard` と `testing` は常に必要。PyTorch を使うプロジェクトでは `pytorch` を追加する。

| スタイル   | 使用タイミング                                        |
| ---------- | ----------------------------------------------------- |
| `standard` | すべての Python ソースファイル — 常に適用。           |
| `testing`  | `tests/` 以下のすべてのテストファイル — 常に適用。    |
| `pytorch`  | 機械学習・深層学習で PyTorch を使用するプロジェクト。 |

## 組み合わせルール

- テスト駆動実装には `tdd` と組み合わせる。
- コードレビュータスクには `review` と組み合わせる。
