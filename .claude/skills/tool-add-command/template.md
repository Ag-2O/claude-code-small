# コマンド定義テンプレート

## 引数なし・スキル呼び出し型

```markdown
---
description: >
  [コマンドの目的を 1〜2 文で説明する。]
---

# 指示

`[skill-name]` スキルを読み込み実行する。

引数がない場合は、現在の会話内容から[操作対象]を特定し、
`[skill-name]` のフェーズ 1 からユーザーと対話しながら進める。
```

## 引数あり・スキル呼び出し型

```markdown
---
description: >
  [コマンドの目的]: $ARGUMENTS
  [補足説明を 1 文で書く。]
---

# 指示

[コマンドの目的]: $ARGUMENTS

[対象] `$ARGUMENTS` で `[skill-name]` スキルを読み込み実行する。
```

## 引数あり・エージェント呼び出し型

```markdown
---
description: >
  [コマンドの目的]: $ARGUMENTS
  [補足説明を 1 文で書く。]
---

# 指示

[コマンドの目的]: $ARGUMENTS

ステップ名または[対象名] `$ARGUMENTS` で `[agent-name]` サブエージェントを呼び出す。
[エージェントへの追加指示があれば記述する。]
```

## フロントマター description の書き方

- 引数なし: 目的を体言止めで 1〜2 文
- 引数あり: 先頭行に `[目的]: $ARGUMENTS`、2 行目以降に補足
- `$ARGUMENTS` はそのまま埋め込む（変数展開はランタイムが行う）
